# Music Manager written in Python

Music catalog using PlayMe to play audio files and streams.<br />
Mp3, Ogg, Flac<br />
<br />
Import Directory: import entire music directory and sub directories.<br />
Add File: add a single audio file.<br />
Add Manually: add a single or multiple audio files and edit meta info.<br />
Edit: edit individual or multiple files meta info.<br />
Delete: delete file(s)
PlayMe: after selecting audio files to be played, click PlayMe to open PlayMe audio player.<br />
Refresh: you guessed it, refresh the database.<br />
Save to File: saves any and all changes made to the music catalog files.<br />
Close: yep you guessed it again, close Music Manager.<br />
Search & Clear Search: self explanitory.<br />
